Add this bot to your Discord server to manage Slippi connect codes cross-server. Use &help to start.

Add or Update A Melee Regional Discord Server

If you'd like to add a regional Discord server not on this list, or if an invite link needs updating, please submit a request below in our Discord server. Please note that we are only accepting servers with 100 or more members at this time, and a submission does not guarantee an inclusion on this list. To avoid including redundant servers, we will only select the most active of the options submitted.
